Educator Dashboard Guide

Use this guide when you are working in the Educator Dashboard.

You are working on the cloud website:

https://rosepch.org

Use the Classroom Hub Device Setup Guide when you need to power on the hub, join its setup hotspot, connect it to Wi-Fi, or find the pairing code.

What You Need

  • Tablet, laptop, or phone with internet
  • Educator email address
  • Classroom Hub pairing code from the physical hub setup page
  • Lesson files, such as images, PDFs, videos, PowerPoints, or APK files
  • Classroom names and student groups

Step 1: Create An Educator Account

  1. Open https://rosepch.org.
  2. Choose Create account.
  3. Enter your full name.
  4. Enter your email address.
  5. Create a password.
  6. Choose Create account.
  7. If the page says to check your email, open your email and confirm the account.
  8. Return to https://rosepch.org and sign in.
Picture to add: screenshot of the Create Account form.

Step 2: Sign In Later

  1. Open https://rosepch.org.
  2. Choose Sign in.
  3. Enter your email address.
  4. Enter your password.
  5. Choose Sign in.

If sign-in fails, check the email and password carefully.

Step 3: Link A Classroom Hub

The physical hub must be powered on and connected to internet Wi-Fi first.

  1. On the physical hub setup page, find the one-time pairing code.
  2. In the Educator Dashboard, open the profile menu.
  3. Choose Link Classroom Hub.
  4. Enter the pairing code.
  5. Submit the code.

After linking, this Educator Dashboard account can send lessons to that hub.

Picture to add: screenshot of the profile menu with Link Classroom Hub.
Picture to add: screenshot of the pairing code form.

Step 4: Unlink Or Replace A Classroom Hub

Use the profile menu when the account needs to stop using a hub.

To unlink without replacing:

  1. Open the profile menu.
  2. Choose Unlink Classroom Hub.
  3. Confirm the unlink.

Lessons and classrooms stay in the Educator Dashboard. The old hub will stop syncing until it is linked again.

To replace with another hub:

  1. Open the profile menu.
  2. Choose Replace Classroom Hub.
  3. Enter the pairing code from the new hub.
  4. Submit the code.
Picture to add: screenshot of the unlink confirmation.

Step 5: Create A Classroom

A classroom is a student group. Each classroom has a 6-character join code.

  1. Open Classrooms.
  2. Enter the classroom name, for example Primary 4 - Village A.
  3. Use the suggested join code, or choose the refresh arrow to make a new one.
  4. Choose Create Classroom.
  5. Save or write down the join code.

Students type the join code in the Student Library to see the right lessons.

Picture to add: screenshot of the Classrooms page with the join code.

Step 6: Create A Lesson

  1. Open Lessons.
  2. Choose Create Lesson.
  3. Enter the lesson title.
  4. Add a description if helpful.
  5. Choose the classroom that should receive the lesson.
  6. Add files by dragging them into the upload area or choosing Browse files.
  7. If the lesson should go to the Classroom Hub, turn on Save to Classroom Hub.
  8. Choose Create Lesson.

Lessons need files and a classroom before they can be useful to students.

Picture to add: screenshot of the Create Lesson window.

Step 7: Upload Files To An Existing Lesson

Accepted file types include:

  • Images
  • PDFs
  • Videos
  • PowerPoint files
  • APK files

To add files:

  1. Open Lessons.
  2. Open the lesson.
  3. Choose Upload Files.
  4. Drag files into the upload area or choose Browse files.
  5. Wait until upload finishes.
  6. Close the upload window.
Picture to add: screenshot of the Upload Files window.

Step 8: Choose Lessons For The Classroom Hub

Only selected lessons are saved to the Classroom Hub.

  1. Open a lesson.
  2. Find the Save to Classroom Hub control.
  3. Turn it on for lessons students need offline.
  4. Turn it off for lessons that should stay only in the Educator Dashboard.

If no hub is linked, this control may be disabled until the hub is linked.

Step 9: Sync Lessons

Syncing tells the linked Classroom Hub to download the selected lessons.

Before syncing, make sure:

  • The Classroom Hub is powered on.
  • The hub is connected to internet Wi-Fi.
  • The educator account is linked to the hub.
  • The lesson has files.
  • The lesson is assigned to the correct classroom.

To sync:

  1. Open Sync.
  2. Review the lessons listed.
  3. Choose Sync.
  4. Wait for the success message.

If sync fails, check that the hub is powered on and online, then try again.

Picture to add: screenshot of the Sync page.

Step 10: Give Students Their Join Code

Students use the Student Library on the Classroom Hub.

Student address:

http://rosehub.local/join

Give students:

  • The Classroom Hub Wi-Fi network name
  • The Student Library address
  • Their classroom join code

The same Classroom Hub can hold more than one classroom. The join code controls which lessons each student group sees.

Updating Lessons Later

When you need to add or change lessons:

  1. Open https://rosepch.org.
  2. Sign in to the Educator Dashboard.
  3. Create or edit lessons.
  4. Upload new files if needed.
  5. Check classroom assignments.
  6. Open Sync.
  7. Choose Sync.
  8. Wait for success before teaching offline.

Troubleshooting

I Cannot Sign In

  • Check the email address.
  • Check the password.
  • If you just created the account, check email for the confirmation message.

I Cannot Link The Hub

  • Make sure the hub is powered on.
  • Make sure the hub is connected to internet Wi-Fi.
  • Make sure you typed the pairing code exactly.
  • If the code was already used, unlink or replace the hub from the profile menu.

I Cannot Sync

  • Make sure a Classroom Hub is linked.
  • Make sure the hub is powered on.
  • Make sure the hub has internet.
  • Make sure each lesson has files.
  • Make sure each lesson is assigned to a classroom.
  • Try Sync again.

Students Cannot See Lessons

  • Make sure students typed the correct classroom join code.
  • Make sure the lesson is assigned to that classroom.
  • Make sure the lesson was saved to the Classroom Hub.
  • Make sure sync completed before class.
