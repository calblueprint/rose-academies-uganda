Rose Portable Classroom Hub: Guide Index

The Rose Portable Classroom Hub system has two setup guides because educators use two different places for two different jobs.

Which Guide Should I Use?

Guide Where you are working Use it for
Classroom Hub Device Setup Guide On the physical Classroom Hub device at rosehub.local/setup Power, hotspot, Wi-Fi, pairing code, linked status, downloading files
Educator Dashboard Guide On the Educator Dashboard at rosepch.org Account, link/unlink hub, classrooms, lessons, uploads, sync, join codes

Important Names

Name What it means
Rose Portable Classroom Hub The full system
Educator Dashboard The website at rosepch.org where educators manage lessons and classrooms
Classroom Hub device The physical hub device that creates the local student library
Classroom Hub Device Setup The local setup page at rosehub.local/setup
Student Library The student-facing library served by the Classroom Hub
Pairing code The one-time code shown by the hub and entered in the Educator Dashboard
Join code The classroom code students type in the Student Library

Simple Rule

Educators manage teaching content in the Educator Dashboard.

The physical hub is set up on the Classroom Hub Device Setup page.

Students use the Student Library after setup and sync are complete.
